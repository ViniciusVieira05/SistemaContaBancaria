# SistemaContaBancaria

Este projeto foi desenvolvido com o objetivo exclusivo de estudo e preparação para a prova da disciplina de Programação Orientada a Objetos (POO) do curso de Análise e Desenvolvimento de Sistemas (ADS).

O sistema simula o funcionamento básico de um banco, utilizando conceitos fundamentais da POO, como:

° Herança

° Encapsulamento

° Polimorfismo

° Classes abstratas

° Sobrescrita de métodos

° Entrada de dados com Scanner

## Funcionalidades do Projeto

Cadastro de Conta Corrente e Conta Poupança

Depósito de valores

Saque com validação de saldo e limite

Aplicação de juros na Conta Poupança

Exibição dos dados da conta

## Objetivo

O principal intuito deste projeto é reforçar os conceitos de Programação Orientada a Objetos na prática, servindo como material de estudo para a prova e também como apoio para outros estudantes que estejam aprendendo Java e POO.
